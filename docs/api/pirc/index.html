Overview

The Pirouette compiler implements higher-order typed functional choreographies, allowing developers to write distributed protocols as single, unified programs that are then compiled to individual endpoint implementations.

The compiler translates choreographies (global descriptions of multi-party protocols) into executable code for each participant through a multi-stage pipeline: parsing, type checking, endpoint projection, and code generation.

Pirouette supports multiple communication backends (HTTP, shared memory) and generates standalone programs that preserve the protocol guarantees specified in the choreography.
