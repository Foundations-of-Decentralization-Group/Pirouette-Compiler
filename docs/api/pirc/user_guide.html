Overview

Pirouette is a compiler for choreographic programming - a paradigm where you write distributed protocols as a single, unified program describing all participants' interactions from a global perspective.

The compiler automatically:

  • Projects the choreography onto individual endpoint programs
  • Ensures communication safety and protocol correctness
  • Generates executable code with your choice of communication backend

Key Benefits:

  • Write distributed protocols once instead of per-endpoint
  • Compile-time guarantees of deadlock freedom
  • Automatic endpoint generation preserves choreographic properties

Quick Start

Install dependencies:

  opam install . --deps-only --with-test

Build pirc:

  dune build

Build and execute:

  dune exec pirc -- [options] <file>

Once installed, run pirc directly:

  pirc [options] <file>

Example:

  cd examples
  dune exec pirc -- -ast-dump json ex2.pir

This will parse the file and dump the ASTs to the current directory in pretty-printed format.

Command-Line Options

  • -ast-dump pprint: Pretty print the AST
  • -ast-dump json: Output AST in JSON format
  • -backend shm: Use shared memory communication
  • -backend http: Use HTTP communication

Libraries

The http_pirc library provides the HTTP-based runtime for endpoint communication:

  • Config_parser - Parse YAML configuration files mapping locations to HTTP addresses
  • Send_receive - Send and receive messages between endpoints over HTTP
